# هَون (Hawn)

## نبذة عن المشروع

هَون هي منصة رقمية تعتمد على تقنيات الذكاء الاصطناعي، وتهدف إلى تبسيط المعلومات الطبية وتحويل المحتوى الطبي المعقد إلى ملخصات واضحة وسهلة الفهم.

تركز المنصة على تحسين التواصل بين الطبيب والمريض من خلال تنظيم المعلومات الطبية المهمة مثل التشخيصات والأدوية والتعليمات العلاجية بطريقة سهلة القراءة والاستيعاب، مما يساعد المستخدمين على متابعة خططهم العلاجية وفهم المعلومات الصحية بشكل أفضل.

---

## المميزات الرئيسية

- تحليل النصوص الطبية باستخدام الذكاء الاصطناعي.
- تنظيم وعرض المعلومات الطبية بشكل مبسط وواضح.
- تصنيف التشخيصات والأدوية والتعليمات العلاجية.
- واجهة مستخدم سهلة الاستخدام وسريعة الوصول للمعلومات.
- دعم مشاركة وحفظ الملخصات الطبية.

---

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- Noha 2.0
- Vercel

اعتمد المشروع على نموذج Noha 2.0 المطور من قبل شركة عِلم، والذي يتميز بقدرات متقدمة في فهم ومعالجة اللغة العربية. ساهم ذلك في تحليل النصوص الطبية واستخلاص المعلومات المهمة وتحويلها إلى ملخصات منظمة وسهلة الفهم، مما عزز من كفاءة ودقة الحل المقترح.



---

## نوع المشروع

تم تطوير المشروع ضمن هاكثون ذكاثون المقدم من شركة عِلم، حيث ركز المشروع على توظيف تقنيات الذكاء الاصطناعي ومعالجة اللغة العربية لتقديم حل مبتكر في المجال الصحي.

---
طريقة التشغيل
يمكنك استعراض المشروع وتشغيله بإحدى الطرق التالية:

1. التشغيل السحابي (Live Demo)
يمكنك الوصول للمنصة مباشرة عبر الرابط الرسمي على Vercel: https://hawnthakathon.vercel.app

2. التشغيل المحلي (Local Setup)
في حال رغبتك في تشغيل المشروع على جهازك الشخصي، اتبع إحدى الطريقتين:

الطريقة السريعة:     1. قم بتحميل المجلد وفك الضغط عنه.     2. افتح ملف index.html مباشرة باستخدام أي متصفح ويب (Chrome أو Edge).

الطريقة البرمجية (للمطورين):     1. افتح مجلد المشروع باستخدام برنامج Visual Studio Code.     2. اضغط على زر "Go Live" في الشريط السفلي لتشغيل الموقع عبر إضافة Live Server.     3. سيتم فتح الموقع على العنوان المحلي http://127.0.0.1:5500.

ملاحظة: يتطلب التشغيل (سواء السحابي أو المحلي) وجود اتصال فعال بالإنترنت لضمان قدرة المنصة على الاتصال بـ API نماذج عِلم ومعالجة النصوص طبياً بشكل فوري.
